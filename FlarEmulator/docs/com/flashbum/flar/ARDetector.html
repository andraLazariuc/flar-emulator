Classpublic class ARDetector
InheritanceARDetector flash.events.EventDispatcher

The ARDetector is a manager for the FLARToolKit and helps facilitate the setup, configuration, and detection of markers while using the FLARToolKit's underlining core classes. This class is a modification of an example on http://saqoosha.net/en/flartoolkit/start-up-guide

detectMarker()method 
public function detectMarker(threshold:int = 90, confidence:Number = .5):Boolean

This validates if we have found a marker from the set src (raster). It returns a true/false based on the supplied values. You must supply a source for this to work correctly.

Parameters

threshold:int (default = 90) — The threshold value to be used for detecting the marker.
 
confidence:Number (default = .5) — This is how confident the detector is that it found a marker.
